| :--               | :--      |

| **Description**    | Commit stage test coverage on source (Excluding tests) must be greater than 50%. |
| **Maturity Level** | [Level 4](/LEVELS.html#level-4) |
| **Possible Implementations** | You can use Sonarqube to set a coverage gate. |
